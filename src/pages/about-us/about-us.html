About Mobile App Find Your Nearest Filling Station

CEYPET FILLUP MATE enables app users to locate nearest Fuel stations for the user selected fuel type from over 2000 of island wise fuel stations. App enables you to check availability of your required fuel type before visit to the fuel station. Even, direct you to the selected fuel station by showing directions. Discover each fuel stations available facilities such as air filling facility, N2 gas facility, VISA card payments, ATM, Washrooms, and super markets with just a one click.
Actually this is a beta version. We will upgrade & add more user friendly features to this app as soon as possible! This will be instantly published to all mobile platforms housing the app, namely : iphones, ipads, android phones, windows phones.

Help?

If it takes time too much when it is loading check your connection speed and when it doesn’t show the map enable location service in your device. If you have any other problems with this application, you can contact us. This application contains default Map with all filling stations. You can enable night mode on this map and you can select filling stations on map with filling types. FILTER is another option which can select filling stations and other facilities. Using COMPLAINTS option user can contact directly with Ceylon Petroleum Corporation.
